Falcon BMS briefing na mobilnom/tabletu

Objavljeno: Pet, 29 Jan 2021

Pod: Stvari

Tagovi: falcon

Prevodi: en

Ako ste igrali Falcon BMS (ili druge simulatore) znate koliko je korisno imati informacije o letu uvijek pristupačne. Pravi piloti koriste sveske svezane oko noge. Mi možemo koristiti mobilni/tablet i sačuvati dosta papira. Ja sam prije primitivno uslikao s mobilnim Briefing stranicu u igri. Bolji način je eksportovati Briefing fajl i otvoriti ga na mobilnom.

U suštini radi ovako: HTTP server koji servira direktorij u kojem se nalaze Briefing fajlovi, koji onda otvorimo pomoću internet preglednika na mobilnom.

Na Linux distribucijama je jednostavno s Pythonom servirati direktorij (python -m http.server). Pošto mi je overkill na Windowsu instalirati samo Python zbog toga, našao sam MiniWeb. Naravno možete koristiti koji god HTTP server hoćete, ali je ovaj pogodan jer je minimalističan.

Nakon toga napravite jednostavni Batch fajl u kojem pokrećete server zajedno s Falcon BMS igrom.

@ECHO OFF
start "Briefing" "C:\locakija_miniweba\miniweb.exe" -r "C:\Program Files\Falcon BMS 4.35\User\Briefings"
start /b "" "C:\Program Files\Falcon BMS 4.34\Bin\x64\Falcon BMS.exe"

U Falcon BMS postavkama imate par opcija za brifinge. Najbitnije je da izaberete "Briefing output to File", što je logično. Osim toga ja koristim "Append new Briefings" opciju, jer želim imati zaseban fajl za svaku misiju. Također predlažem "HTML Briefings" opciju za ljepši izgled.

Pokrenite igru (preko Batch fajla), otvorite kampanju, pa onda Briefing prozor i kliknite Print u gornjem desnom uglu. To će snimiti fajl u direktorij (obično Falcon BMS 4.35\User\Briefings). Otvorite u internet pregledniku adresu gdje HTTP servira (npr. http://192.168.1.100:8000), piše u prozoru MiniWeb programa). Kliknite na fajl koji vam treba i to je to.

Osim HTTP servera ja u Batch fajlu pokrećem i ostale stvari koje su korisne: Opentrack za praćenje pokreta glave i VoiceMacro za glasovne komande.
